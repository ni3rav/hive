## Base URL

All API requests are made to:

```
https://api.hivecms.online/api/public/v1
```

> **Note**: The API is versioned. `v1` is the current stable version. Both `/v1/` and the legacy format (without version) are supported for backward compatibility, but using `/v1/` explicitly is recommended.

## Authentication

Replace `{API_KEY}` in each URL with the key provided to you (or you can issue one from your workspace). Keep it privateâ€”even though it's read-only, it maps directly to your workspace.

## Headers

None required. The public API is configured for simple GET requests, so you can call it straight from the browser or curl without setting anything special.
