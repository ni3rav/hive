Use this section when you need raw endpoint details.

For most integrations, prefer the SDK guides in `/docs/*`.

## Base URL

```text
https://api.hivecms.online/api/public/v1/{API_KEY}
```

## Endpoints

- `GET /posts`
- `GET /posts/{postSlug}`
- `GET /tags`
- `GET /categories`
- `GET /authors`
- `GET /stats`
